数据结构与算法

2017-08-08 11:23:44

对数据结构和算法的总结与学习记录。

学习笔记

数据结构，是解决数据按一定规则快速存储、读取等操作的方法。比如链式结构，当前元素附属信息包含上下元素信息，方便链式使用；hash结构，将key-value，以键的hash值，进行映射，以解决快速读取和查找，但无法排序；等……

算法，查找和排序。因为数据结构的变化及衍变，查找算法基本不需要太复杂，除非是多维数组或对不同字符编码等进行查找。

查找算法

主要为：二分查找和顺序查找
@param $arr 被查找数组
@param $low 被查找数组的最小范围值，默认为0
@param $high 被查找数组的最大范围值，默认为数组长度
@param $k 需要查找值

function bin_sch($arr,$low,$high,$k){
if($low<$high) {
$mid = ($low+$high) / 2;
if($k < $arr[$mid]){
return bin_sch($arr,$low,$mid-1,$k);
} elseif ($k > $arr[$mid]) {
return bin_sch($arr,$mid+1,$high,$k);
} else {
return $mid;
}
}
return -1;
}

排序算法

排序算法成为主要算法，有：基本排序：交换排序（冒泡、快速），选择排序（选择、堆排序），插入排序（插入、希尔），归并排序，基数/桶排序。

  • 冒泡算法

function bubble_sort($arr){
$count = count($arr);
if($count<=1) return false;
for($i=0;$i<$count;$i++){
for($j=$count-1;$j>$i;$j–){
if($arr[$j]<$arr[$j-1]){
$tmp = $arr[$j];
$arr[$j] = $arr[$j-1];
$arr[$j-1] = $tmp;
}
}
}
return $arr;
}

  • 快速排序

function quick_sort($arr){
if(count($arr)<=1) return $arr;
$key = $arr[0];
$left = [];
$right = [];
for($i=1;$i<count($arr);$i++){
if($arr[$i] <= $key) {
$left[] = $arr[$i];
} else {
$right[] = $arr[$i];
}
$left = quick_sort($left);
$right = quick_sort($right);
return array_merge($left,$right);
}

  • 选择排序

原理：将数组最小值与数组第一位置置换，再对剩余值循环最小值与第二位置置换，直到数组长度为1。

function selectSort($arr){
$len = count($arr);
if($len<=1) return false;
for($i=0;$i<$len;$i++){
$p=$i;
for($j=$i+1;$j$arr[i]){
$p = $j;
}
}
if($p!=$i) {
$tmp = $arr[$p];
$arr[$p] = $arr[$i];
$arr[i] = $tmp;
}
}
return $arr;
}
